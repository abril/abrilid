Abril ID

Abril ID é um app crossbroser desenvolvido para as Marcas Abril SA contectarem-se aos seus usuarios.

Version 3

Widget - Abril ID

Atravez dos widget do Abril ID seu site terá mais uma ferramenta para interagir com seus usuários de uma forma mais tranquila, novos usuários podem optar por se cadastrar no Abril ID ou utilizar uma contas como do facebook, google ou twitter.

Newsletter

Para disponibilizar newsletter para seus usuários o Abril ID conta com uma interface rápida e simples de implementação onde com 5 linhas de código seus site poderá prover uma newsletter como essa do casa.com.br

Promoções

Flexivel, Rápido e Customizável o Abril ID pode ter a cara do seu site, da sua promoção, estar na posição desejável e ainda responder a um volume de chamadas por minuto sem demorar e o publico da capricho não deixa duvida disso.

Topo Abrill ID

O Topo é o mais novo componente do Abril ID. Tem uma interface limpa podendo ser colocado no topo de site e manter a identidade do site e apresentar uma saudação marcante para seus usuários

O que é Widget

Widget é um componente de interface gráfica que "flutua" sobre uma determinada região navegada pelo usuário.

Vantagens

O Abril ID funciona via facebook, Twitter, google. Disponibiliza newsletter por marcas e Tem integração com o SocialCore a mais nova Abril para RedesSociais
