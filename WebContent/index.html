Abril ID

Abril ID é um serviço de fácil instalação que permite qualquer site da Abril indentificar seus usuários e oferecer a eles conteúdo e funcionalidades exclusivas.

Version 3

Widget - Abril ID

Ao instalar o widget, o site já ganha várias funcionalidades, tais como login via Facebook, Twitter ou Google, gerenciamento de senha, emails, dados de perfil.

Newsletter

Com apenas 5 linhas de código, seu site também poderá oferecer uma maneira simples e rápida para o usuário gerenciar o acesso às newsletters, assim como o casa.com.br já fez.

Promoções

Flexivel, rápido e customizável o Abril ID pode ter a cara do seu site, da sua promoção, estar na posição que você desejar, tudo isso sem atrapalhar a performance do seu site.

Topo Abrill ID

O Topo é o mais novo componente do Abril ID. Tem uma interface limpa, mantém a identidade do site e apresenta uma saudação marcante para seus usuários.

O que é Widget

Widget é um componente externo que estende as funcionalidades do seu site, sem que seja necessário um processo de implementação ou instalação complicado.

Vantagens

O Abril ID funciona via facebook, Twitter ou Google. Disponibiliza newsletters por marcas e tem integração com o SocialCore, serviço que permite aos site implementarem funcionalidades sociais.

Documentação

Este site tem o intuito de explicar tudo sobre o Abril ID e fazer com que a instalação e customização seja bem mais fácil
