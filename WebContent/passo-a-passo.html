Passo à Passo

1º Passo - Instalando o Abril ID

Para instalar o Javascript do Abril ID é muito simples basta inserir a tag widget.js, o ideial é que a tag esteja contida no final da página que você quer aplicar o componente do Abril ID

Atenção para realizar testes de implementação devemos adicionar qa a frente da url como no exemplo abaixo: 
  1. <script src="http://id.abril.com.br/javascripts/widgets.js">
  2. </script>

2º Passo - Configurando o Abril ID

Para configurar o Abril ID no seu site ele deverá possuir no mínimo as seguintes configurações que são obrigatórias, que são o produto: que deve conter a marca que será implementada(no caso o nome da revista) no Abril Id e deverá possuir um container: que no caso é nome da div que será implementada. Como no exemplo a seguir.

Atenção Esse script deverá estar na página na qual você quer implementar o Abril ID. 
  1. <script type="text/javascript">
  2.    var configuracao = {
  3.      produto:'ID_DO_SEU_SITE',
  4.      container: 'ID_DA_DIV'
  5.    };
  6.  </script>
Exemplo Prático Esse exemplo evidência uma forma prática e objetiva de implementação do widget. 
  1. <script type="text/javascript">
  2.    var configuracao = {
  3.      produto:'VEJA',
  4.      container: 'abrilID'
  5.    };
  6.  </script>

Observação: Perceba que no scrpit acima temos na linha 3 o id do seu site, e na linha 4 temos o nome da div que iremos aplicar o widget do abril id ou qualquer outro componente que você queira chamar na tela.

produto: Repare que nessa tag devo inserir o ID do Site.

container: Repare que nessa tag devo inserir o nome da Div que eu quero que componente apareça.

3º Passo - Instanciando o Abril ID

Para instanciar o Abril ID deveremos inserir o código abaixo na sua página para termos o widget.

AtençãoEsse é um exemplo teórico para evidenciar uma instancia de widget. 
  1. <script type="text/javascript">
  2.   var exemplo = new AbrilId();
  3.   exemplo.initialize(configuracao);
  4. </script>
Exemplo Prático Esse exemplo evidência uma forma prática e objetiva de implementação do widget. 
  1. <script type="text/javascript">
  2.   var widgets = new AbrilId();
  3.   widgets.initialize(configuracao);
  4. </script>

4º Passo - Criar uma Div para implementação do widget

Para que a implementação do Abril ID seja completa precisamos criar uma div dentro do body que ficará responsavel pela chamada do widget do Abril ID.

AtençãoEsse é um exemplo prático para evidenciar a criação da div. 
  1. <body>
  2. <div id='abrilID'>
  3. </div>
  4. </body>

5º Passo - Realizando a chamada do Widget Abril ID com um link

Para invocar o widget na página utilizaremos um link para a execução do mesmo colocando dentro da div criada anteriormente

Exemplo PráticoEsse é um exemplo prático para evidenciar a chamada do widget dentro de um link. 
  1. <a href="javascript: void(0);" onclick="widgets.login.render();">login</a>
