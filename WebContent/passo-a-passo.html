Atenção Essa sessão passo-a-passo não subistitu a leitura da documentação do abril ID. Sua função é ajudar webmaster realizarem as primeiras implementações.

1º Passo - Instalando o Abril ID

Para instalar o Javascript do Abril ID é muito simples. Basta incluir a linha abaixo em todas as páginas onde o Abril ID for necessário.

O ideial é que está linha esteja contida no final da página que você quer aplicar o componente do Abril ID
Atenção para realizar testes de implementação seu ambiente de desenvolvimento deve ter acesso ao nosso anbiente de homologação 
    <script src="http://qa.id.abril.com.br/javascripts/widgets.js"></script>

2º Passo - Configurando o Abril ID

Para o Abril ID funcionar em seu site é necessario incluir sua configuração.

Existem duas formas de configurar o abril ID a forma simplicifada onde você só precisa configurar dois parametros:

Produto: que deve ser um nome/ID para sua marca.

Container: que deve ser o id de de um elemento div dentro de sua pagina.

A forma avançada que está detalhada em outra sessão. Segue exemplo de implementação escrito em json

Caso ainda não tenha um nome/ID deverá ser solicitado para a equipe de desenvolvimento do Abril ID.

Alerta Esse script deverá estar na página na qual você quer implementar o Abril ID. 
  1. <script type="text/javascript">
  2.    var configuracao = {
  3.      produto:'nome/ID_da_seu_site',
  4.      container: 'nome_da div_gerado_para_sua_implementação'
  5.    };
  6.  </script>

3º Passo - Inicializando o Abril ID

Para instanciar o funcionamento do Abril ID devemos criar um objeto javascript e declarar o função .initialize() passando a objeto json configuração do Passo 2 para o Abril ID.

  1. <script type="text/javascript">
  2.   var abrilid_teste = new AbrilId();
  3.   abrilid_teste.initialize(configuracao);
  4. </script>

4º Passo - Criar uma DIV para implementação do abril ID

Para que a implementação do Abril ID seja completa precisamos criar uma div dentro da pagina do seu site com um id que deve ter o mesmo nome usado no Passo 2 no campo compoente. Essa div ficará responsavel por apresentar Abril ID na tela por isso é importante posicionar essa div em um ponto extrategico na pagina do seu site .

  1. <body>
  2. <div id='abrilID'>
  3. </div>
  4. </body>

5º Passo - Chamando do Abril ID atravez de um link

Para invocar o Abril ID na página utilizaremos um link para a realizar chamada do widget de login do Abril ID 

  1. <a href="javascript: void(0);" onclick="widgets.login.render();">login</a>
