1º Passo - Instalando o Abril ID

Para instalar o Javascript do Abril ID é muito simples. Basta incluir a linha abaixo em todas as páginas onde o Abril ID for necessário.

O ideial é que está linha esteja contida no final da página que você quer aplicar o componente do Abril ID 
    <script src="http://qa.id.abril.com.br/javascripts/widgets.js"></script>
Atenção para realizar testes de implementação seu ambiente de desenvolvimento deve ter acesso ao nosso anbiente de homologação

2º Passo - Configurando o Abril ID

Para o Abril ID funcionar em seu site é necessario incluir sua configuração.

Existem duas formas de configurar o abril ID a forma simplicifada onde você só precisa configurar dois parametros:

Produto: que deve ser um nome/ID para sua marca.

Container: que deve ser o id de de um elemento div dentro de sua pagina.

A segunda forma é está mais detalhada em outra sessão.

Segue exemplo de implementação escrito no formato json

Caso ainda não tenha um nome/ID deverá ser solicitado para a equipe de desenvolvimento do Abril ID.

  1. <script type="text/javascript">
  2.    var configuracao = {
  3.      produto:'nome/ID_da_seu_site',
  4.      container: 'nome_da div_gerado_para_sua_implementação'
  5.    };
  6.  </script>

3º Passo - Inicializando o Abril ID

Para instanciar o funcionamento devemos criar um objeto javascript do Abril ID e declarar o função .initialize() passando a objeto json configuração do Passo 2 para o Abril ID.

  1. <script type="text/javascript">
  2.   var abrilid_teste = new AbrilId();
  3.   abrilid_teste.initialize(configuracao);
  4. </script>

4º Passo - Criar uma DIV para implementação do abril ID

Para que a implementação do Abril ID seja completa precisamos criar uma div dentro da pagina do seu site com um id que deve ter o mesmo nome usado no Passo 2 no campo compoente. Essa div ficará responsavel por apresentar Abril ID na tela por isso é importante posicionar essa div em um ponto extrategico na pagina do seu site .

  1. <body>
  2. <div id='abrilID'>
  3. </div>
  4. </body>

5º Passo - Chamando do Abril ID atravez de um link

Para invocar o Abril ID na página utilizaremos um link para a realizar chamada do widget de login do Abril ID 

    <a href="javascript: void(0);" onclick="widgets.login.render();">login</a>

Componentes Abril ID

O Abril ID atualmente possui 3 tipos diferente de componentes Widget, Include e Topo. Cada um deles possuim funcionalidades compartilhadas e funcionalidades especificas como descrito na sessão de documentação > componentes.

A seguir segui como configurar o parametro para instanciar o Abril ID como widget, include ou topo.

Configurando widget



Info: Caso não seja declarado o campo componente por padrão o componente carradado é sempre o widget

  1. <script type="text/javascript">
  2.   var configuracao = {
  3.     produto:'nome/ID_da_seu_site',
  4.     container: 'div_exempl'
  5.     componente: Componente.WIDGET
  6.   };
  7. </script>

Configurando include



  1. <script type="text/javascript">
  2.   var configuracao = {
  3.     produto:'nome/ID_da_seu_site',
  4.     container: 'div_exempl'
  5.     componente: Componente.INCLUDE
  6.   };
  7. </script>

Configurando topo  new

Topo é o mais novo componente do Abril ID e ele contem três opções diferente de formado chados de mini, pequeno e medio 





  1. <script type="text/javascript">
  2.   var configuracao = {
  3.     produto:'nome/ID_da_seu_site',
  4.     container: 'div_exempl'
  5.     componente: Componente.TOPO,
  6.     topo: 'mini'
  7.   };
  8. </script>




  1. <script type="text/javascript">
  2.   var configuracao = {
  3.     produto:'nome/ID_da_seu_site',
  4.     container: 'div_exempl'
  5.     componente: Componente.TOPO,
  6.     topo: 'pequeno'
  7.   };
  8. </script>




  1. <script type="text/javascript">
  2.   var configuracao = {
  3.     produto:'nome/ID_da_seu_site',
  4.     container: 'div_exempl'
  5.     componente: Componente.TOPO,
  6.     topo: 'medio'
  7.   };
  8. </script>

Redes Sociais

Por padrão o login via redes sociais (Facebook, Twitter ou Google Accounts) estão habilitados pelo Abril ID .

Este configuração permite a ativação/desativação do login via redes sociais.



Exemplos:

provedores: ['facebook','twitter'] //facebook e twitter

provedores: ['google'] //fapenas google

provedores: ['google','twitter'] //google e twitter

provedores: [] //nenhum provedor

  1. <script type="text/javascript">
  2.   var configuracao = {
  3.     produto:'nome/ID_da_seu_site',
  4.     container: 'div_exempl'
  5.     provedores: ['facebook', 'twitter']
  6.   };
  7. </script>

Configurando Layout do Abril ID

Customizar Abril ID

O Abril ID é customizavel e permite a configuração de layout para ficar adequado ao seu site.

Para configurar esses parametros você tem a possibilidade de ajustar 6 item:

  1. Layout Geral paramentros general
  2. Linhas parametro inner-box
  3. Botões parametro button
  4. Mensagem de erros parametro error
  5. Logo Abril parametro header
  6. Layout topo parametro topo

Atenção:

a configuração completa encontra-se na sessão documentação > layout

Layout Geral



  1. var configuracao = {
  2.   produto:'nome/ID_da_seu_site',
  3.   container: 'div_exempl',
  4.   estilo:{ 
  5.     'general': {
  6.       'background-color':'#2E0F4D', //Cor de Fundo
  7.       'border-color':'#CCCC43', //Cor da borda
  8.       'border-width':'5', //largura da borda
  9.       'text-color':'#CCCC43', //Cor do Texto
  10.       'inner-border-color':'#7A83FF' //Cor de Fundo Caixa
  11.     },
  12.     ...
